# Métodos Cuantitativos en Investigación

Repositorio académico oficial del curso Métodos Cuantitativos en Investigación del Doctorado en Tecnologías de la Información de la Universidad Nacional Abierta y a Distancia (UNAD).

## Objetivo

Este repositorio centraliza la documentación, recursos académicos y la matriz documental utilizada durante el desarrollo del curso.

## Contenido

- Matriz documental
- Lecturas recomendadas
- Bases de datos
- Cronograma de actividades
- Recursos complementarios

## Actualización

Los contenidos se actualizan de manera permanente durante el periodo académico.
